Erste Schritte: Erstellen der Projekt-Struktur mit Vite

  1. npm create vite@latest    ->    erstellt eine neue Vite Projekt.
  2. ? Project name: » react-projekt    ->    das Projekt einen Namen mit „react-projekt“ geben.
  3. React auswählen unter "Typ" von unserem Vite Projekt.
  4. Javascript auswählen als die benutzte Sprache für unseren Code.
  5. cd react-projekt    ->    im Terminal in das Projekt navigieren.
  6. Die folgenden Ordner und Dateien löschen oder modifizieren:
    • Public Ordner löschen
    • Assets Ordner (der im src Ordner befindet) löschen
    • App.css Datei löschen
    • In App.jsx alle Importe löschen und das Return leeren.
    • Index.css leeren
