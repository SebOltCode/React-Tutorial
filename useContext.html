React-Hooks: useContext

Der useContext Haken wird verwendet, um Werte aus einem React-Kontext zu konsumieren. Der Kontext bietet eine Möglichkeit, Daten durch den Komponentenbaum zu leiten, ohne auf jeder Ebene Argumente (props) manuell übergeben zu müssen.

Verwendungsbeispiel von useContext der den Authentifizierungsstatus eines Benutzers speichert.

import React, { createContext, useContext, useState } from 'react';
            const AuthContext = createContext();
            
            export const AuthProvider = ({ children }) => {
              const [isAuthenticated, setIsAuthenticated] = useState(false);
            
              const login = () => {
                setIsAuthenticated(true);
              };
            
              const logout = () => {
                setIsAuthenticated(false);
              };
            
              return (
                <AuthContext.Provider value={{ isAuthenticated, login, logout }}>
                  {children}
                </AuthContext.Provider>
              );
            };
            
            export const useAuth = () => {
              return useContext(AuthContext);
            };

Wir haben einen Kontext AuthContext mit createContext erstellt und eine AuthProvider-Komponente bereitgestellt. Die AuthProvider-Komponente umhüllt ihre untergeordneten Elemente mit dem Kontextanbieter und enthält Funktionen zum An- und Abmelden. 

import React from 'react';
            import { useAuth } from './AuthContext';
            
            const AuthStatus = () => {
              const { isAuthenticated, login, logout } = useAuth();
            
              return (
                <div>
                  <p>User is {isAuthenticated ? 'logged in' : 'logged out'}</p>
                  <button onClick={login}>Login</button>
                  <button onClick={logout}>Logout</button>
                </div>
              );
            };
            
            export default AuthStatus;

Der useAuth Hook wird verwendet, um auf die vom AuthContext bereitgestellten Werte zuzugreifen. Die AuthStatus-Komponente zeigt den Anmeldestatus des Benutzers an und stellt Schaltflächen zum An- und Abmelden bereit.
