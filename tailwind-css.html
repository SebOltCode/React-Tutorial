TAILWIND CSS ADDON

                    1. npm install -D tailwindcss   
                    2. npx tailwindcss init 
                    3. tailwind.config.js modifizieren mit folgende Inhalt :  
                    /** @type {import('tailwindcss').Config} */ 
                    module.exports = { 
                      content: [ 
                        "./src/**/*.{js,jsx,ts,tsx}", 
                      ], 
                      theme: { 
                        extend: {}, 
                      }, 
                      plugins: [], 
                    } 
                    4. Füge die folgenden Code-Zeilen in index.css : 
                                           @tailwind base; 
                    @tailwind components; 
                    @tailwind utilities;

                            ***** PRETTIER ADDON FÜR REACT ***** 
                    1.  npm install --save-dev --save-exact prettier >>>> Prittier 
                    installieren  
                    2.  node --eval "fs.writeFileSync('.prettierrc','{}\n')" >>>> config 
                    Datei erstellen. 
                    3.  node --eval "fs.writeFileSync('.prettierignore','# Ignore 
                    artifacts:\nbuild\ncoverage\n')"  >>>>  .prettierignore Datei 
                    erstellen .  
                    4.  npx prettier . –write  >>>> alles mit Prittier Addon 
                    formatieren.
